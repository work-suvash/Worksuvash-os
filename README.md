# Worksuvash OS

A modern web-based operating system built with HTML, CSS, and JavaScript.

## Features

- 🖥️ **Desktop Environment** - Full-featured desktop with taskbar, start menu, and desktop icons
- 📁 **File Manager** - Browse and manage your virtual file system
- 💻 **Terminal** - Command-line interface with basic commands
- 📝 **Notepad** - Simple text editor
- 🔢 **Calculator** - Functional calculator application
- 🌐 **Web Browser** - Browser interface (demo)
- ⚙️ **Settings** - System settings and configuration
- 🪟 **Window Manager** - Draggable, resizable windows with minimize/maximize/close
- ⏰ **System Tray** - Clock and system information

## Running the OS

### Quick Start

1. Clone the repository
2. Run the server:
   ```bash
   npm start
   ```
   or
   ```bash
   python3 -m http.server 8000
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

### Features Overview

**Desktop**
- Double-click desktop icons to launch applications
- Click the Start button to access the application menu
- View the current time in the taskbar

**Window Management**
- Drag windows by their title bar
- Use the window controls to minimize, maximize, or close
- Click any window to bring it to the front

**Applications**
- **File Manager**: Browse the virtual file system with Documents, Pictures, and Downloads folders
- **Terminal**: Execute commands like `help`, `echo`, `date`, `ls`, `whoami`, and `clear`
- **Notepad**: Create and edit text documents
- **Calculator**: Perform basic arithmetic operations
- **Web Browser**: Demo browser interface
- **Settings**: View system information and settings

**Shutdown**
- Click the Start button and select "Shutdown" to power down the system

## Technology Stack

- Pure HTML5, CSS3, and JavaScript
- No external dependencies
